
PROFESSIONAL GITHUB-READY JEWELRY WEBSITE

HOW IT WORKS
-------------
Products are stored permanently in:

/data/products.json

To add products:
1. Open products.json
2. Copy existing structure
3. Add product object
4. Save file
5. Upload to GitHub
